## Cara Membuka Source Code

Untuk membuka source code, dapat digunakan text editor (seperti NOTEPAD) atau dapat juga digunakan Integrated Development Environment (IDE) seperti [Visual Studio Code](https://code.visualstudio.com/download).

## Cara Mengkompilasi Source Code dan Menjalankannya

Untuk mengkompilasi source code, dibutuhkan compiler seperti [GCC](https://gcc.gnu.org/) yang telah terinstall di device anda.

Setelah GCC terinstall di device anda, buka terminal dan buka lokasi dimana file `soal1.c` berada.

```
cd "C:\Lokasi\Source\Code\Aqeel"
```

Lalu jalankan GCC dalam terminal untuk mengkompilasi source code.

```
gcc .\soal1.c
```

Setelah kompilasi berhasil, akan muncul file `a.exe` yang bisa dijalankan di terminal.

```
.\a.exe
```

## Cara Membuka DFD dan Flowchart

Bukalah web [draw.io](https://app.diagrams.net/), tekan tombol `Open Existing Diagram`, cari folder lokasi `DFD dan Flowchart Praktikum PMP Modul 5.drawio`. Disini akan terlihat beberapa tab untuk tiap DFD dan Flowchart yang telah dibuat.
